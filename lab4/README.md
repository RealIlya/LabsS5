# Telegram Task Manager Bot

Это простой, но функциональный Telegram-бот для управления списком личных задач. Он использует Node.js и SQLite для хранения данных.

## Возможности

- **Добавление задач**: Добавляйте новые задачи в свой список.
- **Просмотр задач**: Просматривайте список невыполненных дел в удобном формате с постраничной навигацией.
- **Управление задачами**: Отмечайте задачи как выполненные или удаляйте их прямо из чата с помощью интерактивных кнопок.
- **Безопасность и изоляция данных**: Хотя все данные хранятся в общей базе, доступ к ним строго разграничен на уровне запросов. Ни один пользователь не может просмотреть или изменить задачи другого.
- **Надежность**: Использование параметризованных запросов для защиты от SQL-инъекций.

## Установка и запуск

Для запуска бота на вашем сервере или локальной машине выполните следующие шаги.

### 1. Установка зависимостей

Установите все npm-пакеты, указанные в `package.json`.

```bash
npm install
```

### 2. Настройка токена

Боту требуется токен для доступа к Telegram Bot API.

1.  Создайте в корневой папке проекта файл с именем `.env`.
2.  Откройте этот файл и добавьте в него следующую строку, заменив `ВАШ_СЕКРЕТНЫЙ_ТОКЕН` на реальный токен вашего бота, полученный от [@BotFather](https://t.me/BotFather):

```
TELEGRAM_BOT_TOKEN=ВАШ_СЕКРЕТНЫЙ_ТОКЕН
```

### 3. Запуск сервера

После установки зависимостей и настройки токена запустите бота командой:

```bash
node bot.js
```

Если все настроено правильно, в консоли появятся сообщения:

```
Подключение к базе данных SQLite успешно установлено.
Бот запущен и готов к работе...
```

Теперь вы можете найти своего бота в Telegram и начать им пользоваться!

## Список команд

- `/start` — Показать приветственное сообщение и список команд.
- `/help` — Показать помощь.
- `/add [текст задачи]` — Добавить новую задачу.
- `/list` — Показать список невыполненных задач.
- `/done [ID]` — Пометить задачу как выполненную (альтернатива кнопкам).
- `/delete [ID]` — Удалить задачу (альтернатива кнопкам).
